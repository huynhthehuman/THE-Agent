# 🤖 CẨM NANG SỬ DỤNG "THE AGENT" DÀNH CHO NHÂN SỰ
*(Tài liệu lưu hành nội bộ giúp nhân sự tận dụng tối đa sức mạnh của AI trong công việc hàng ngày)*

Dưới đây là **Toàn bộ vũ khí** mà nhân sự các phòng ban có thể ngay lập tức sử dụng cùng THE Agent để tăng tốc công việc, giảm sai sót và bảo vệ bản thân khỏi các rủi ro đền bù.

---

## 🟢 1. BỘ TƯỚNG LĨNH COPILOT (CHAT TRỰC TIẾP VỚI AGENT)
Cách dùng: Ở bất kỳ khung chat nào với Agent, chỉ cần gõ đúng Cú pháp Tag tên (+ Câu hỏi/Vấn đề), Agent sẽ tự động hóa rập khuôn tư duy của quản lý để giải quyết cho bạn.

### 🛡️ Dành cho Phòng CS (Chăm sóc Khách hàng)
> Cú pháp: `@cs-copilot [Mô tả tình huống khách chửi/bắt đền]`
- **Chữa cháy khủng hoảng:** Khách hàng chửi bới vì rớt track, Delay, hàng bị nát. Bạn ném nguyên văn đoạn chat của khách vào, Agent sẽ dịch lỗi, tra bộ 13 luật vận hành và **soạn sẵn mẫu tin nhắn mềm mỏng (nhưng cứng rắn về luật)** để bạn copy gửi khách.
- **Duyệt hồ sơ đền bù (Damage Claim):** Quăng tình huống kiện hàng vỡ vào, Agent sẽ check xem khách đã cung cấp đủ 4 loại ảnh bắt buộc chưa, và hàng >25kg có lót xốp không trước khi cho phép bạn xoa dịu khách.

### 📈 Dành cho Phòng Sales
> Cú pháp: `@sale-copilot [Mô tả tệp khách hàng / Yêu cầu báo giá]`
- **Lập báo giá chớp nhoáng (Quotation):** Nhờ Agent tự động móc nối file giá bảng dịch vụ nội bộ để lên một sườn báo giá chuyên nghiệp cho line Yun/HPW.
- **Kịch bản chốt Sale (Solution Selling):** Đưa profile của khách (Ví dụ: "Khách bán móng tay giả trên TikTok"), Agent sẽ trả về các mồi nhử Sale tốt nhất, không cần phải dùng bài "Giảm giá".

### 📦 Dành cho Phòng Ops (Kho / Vận hành)
> Cú pháp: `@usps-fulfillment [Mô tả lỗi trên Google Sheet / Mất Barcode]`
- **Cứu hộ hệ thống Google Sheet:** Khi file chia ca, quét Barcode, tạo PDF A7 bị lỗi, Agent sẽ đưa ngay đoạn code Apps Script vá lỗi để chép vào dập lửa.
- **Hỏi đáp nghiệp vụ:** Hỏi nhanh bất cứ quy trình đóng hộp, điền form TSCA/Lacey act cho kiện hàng gỗ.

### 💰 Dành cho Kế toán & HR
> Cú pháp: `@finance-copilot [Kiện hàng XXX chênh lệch 4kg]` HOẶC `@hr-copilot`
- **Finance Copilot:** Kiểm soát bùng bill, soát cước lệch Dim vs Actual. Check xem có bị UPS truy thu thuế sau 361 ngày không.
- **HR Copilot:** Nhân sự mới vào có thể chat với con này để được Onboarding quy định công ty. Manager dùng nó để tự động chấm điểm bài thi KNL.

## 🛠️ 2. BỘ CÔNG CỤ TOOLS KỸ THUẬT SỐ ĐÃ CÓ SẴN TRONG MÁY

Ngoài việc Chat hỏi đáp, THE Agent đã cài đặt sẵn xuống máy tính các phần mềm và File tự động sau (Chỉ việc dùng, không cần code):

1. **Công cụ Check Label (Mã tự động rà soát tem):**
   - **Nơi chứa:** Thư mục `THE Agent/check label/check_pdf.py`
   - **Tác dụng:** Nhân sự kho có thể chạy file này để máy tự động đọc toàn bộ Text và thông tin giấu trong file PDF nhãn vận chuyển (tránh rủi ro mắt thường nhìn nhầm ký tự).
2. **Hệ thống Ingest Data (Chuyển Data Excel thành tri thức siêu tốc):**
   - **Nơi chứa:** `THE Agent/Legacy SOP/ingest_all.py`
   - **Tác dụng:** Bất cứ khi nào phòng ban có file Báo giá cước / File chính sách mới dạng Excel. Cứ việc ấn chạy tool này, Agent sẽ tự động "nhai" và học thuộc lòng để tư vấn ngay lập tức, không cần tốn thời gian dạy lại cho AI.
3. **Bài thi Test Đánh giá Năng lực (Khung KNL):**
   - **Tên file:** `THE Agent/knl_assessment_test.md`
   - **Tác dụng:** Quản lý lấy file này gửi cho nhân sự đầu tháng. Nhân sự tick chọn hoàn thành 5 tình huống sẽ tự động map ra Level 1 - 4 để xếp lương KPI.
4. **Cẩm nang Vận hành DOCX cho lính mới:**
   - **Tên file:** `THE Agent/Quy_Trinh_Van_Hanh_Toan_Dien_THE.docx`
   - **Tác dụng:** File word cầm tay chỉ việc tất cả các quy trình đi đơn FBA, Hàng khó, Kiện tổng dọn sẵn ra mâm. Mang format chuẩn bảng biểu dễ dọc.
